# C-Codes

**C-Codes** is a collection of basic C programs with well-documented syntax. These codes were written during my preparation for college-level C programming.

## Features

- Well-documented code
- Simplified approach
- Provides a good foundation for basics
- Learn while coding scenario

## Technologies

- C programming language and related tools

## Installation and Setup

1. Follow the guide on [Visual Studio Code C++ Setup](https://code.visualstudio.com/docs/cpp/config-mingw) for setting up your development environment.

## Usage

1. Clone the repository using:
   ```bash
   git clone https://github.com/AtharvVatsal/C-Codes.git
   ```
2. Navigate to the project directory:
   ```bash
   cd C-Codes
   ```
3. Open the C files in your preferred code editor and start exploring or modifying the code.

## Code Examples

Here are a few examples of the types of programs included in this repository:

### Example 1: Hello World

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

### Example 2: Simple Calculator

```c
#include <stdio.h>

int main() {
    int a, b, sum;
    printf("Enter two integers: ");
    scanf("%d %d", &a, &b);
    sum = a + b;
    printf("Sum: %d\n", sum);
    return 0;
}
```
